# Linux Foundations & Commands

Linux is a family of open-source UNIX based operating systems. Linux is responsible for powering important systems around the globe, like cloud servers. Due to this, it becomes the playground for Cybersecurity attacks, as most attacks come from environments based off Linux. This is where you may find the majority of your time as a Cybersecurity professional. 

For Linux based learning/labs, Ubuntu was used, which is an OS (Operating System) that runs off Linux, the core kernal. If Linux is the engine, Ubuntu is the car built around that engine.
